## To launch this App, follow these steps:

### 1. Install PHP package

```
composer install
```

### 2. Install NPM package

```
npm install
```

### 3. Copy Example env

```
cp .env.example .env
```

### 4. Generate Application Key

```
php artisan key:generate
```

### 5. Build Application

```
npm run build
```

### 6. Start Application

```
php artisan serve
```
